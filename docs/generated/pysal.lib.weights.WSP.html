pysal.lib.weights.WSP¶

class pysal.lib.weights.WSP(sparse, id_order=None)[source]¶

Thin W class for spreg.

Parameters
sparsesparse_matrix

NxN object from scipy.sparse

id_orderlist

An ordered list of ids, assumed to match the ordering in sparse.

Examples

From GAL information

>>> import scipy.sparse
>>> from pysal.lib.weights import WSP
>>> rows = [0, 1, 1, 2, 2, 3]
>>> cols = [1, 0, 2, 1, 3, 3]
>>> weights =  [1, 0.75, 0.25, 0.9, 0.1, 1]
>>> sparse = scipy.sparse.csr_matrix((weights, (rows, cols)), shape=(4,4))
>>> w = WSP(sparse)
>>> w.s0
4.0
>>> w.trcWtW_WW
6.395
>>> w.n
4
